Name

Cube (fa-cube)

Description

This link needs to be opened with an application.

Use

Use it when you want to represent:

  • These are used to represent "Qubes". These are used in "Q Enterprise Messenger" and contain relevant documentation

Example

Sizing

small (16 x 14 pixels)

large (24 x 28 pixels)

Color

AT&T Gray Highlight (#f2f2f2)


Use AT&T Gray Highlight if you are using the icon as a link that displays all of the previously bookmarked pages. You should use AT&T Gray Highlight will be applied if this link is disabled (see Links).


AT&T Blue (#067ab5)


Use AT&T Blue if you are using the icon as a link that displays all of the previously bookmarked pages. You should use AT&T Blue Highlight when the user hovers over the icon (see Links).


General Placement

This is usually paced to the right of the item that the Qube has documentation for.
