Containers

The body is the entire web page - the container is the part of the page that contains the actual content. With today's wide screens if someone opens their browser full screen web pages would become unusable if they didn't limit their width (about 30 or 35 em is the maximum width for a readable column so you'd need lots of columns of data to actually fill the width of a widescreen monitor with usable content.Far easier to use a container around your content that allows a background to display on either side once the viewport is wider than would allow the content to be readable if it filled the entire width.
