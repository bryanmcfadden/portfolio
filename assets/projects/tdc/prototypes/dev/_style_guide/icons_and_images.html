File Format

The most widely used file formats for images in websites are JPG, PNG and GIF. These formats are supported across all current web browsers; for this reason, you should always use one of these formats when using images in your site.

So, which image format should you use? JPG is typically used for photo images; use it if image quality is less important than file size (i.e., you want small file sizes and you do not mind a lower quality image). On the other hand, PNG and GIF are commonly used for graphics with transparent backgrounds and/or graphics that do not use too many colors. If you do want crisp graphics and do not care much for file size, either of these formats should do the trick. The main difference between GIFs and PNGs is that GIFs can be animated and cannot support over 256 colors.

JPG/JPEG

This format is perfect for pictures with lots of colors (over 256 colors) and is the most well-known format. In general, JPGs are also great if you want to create an image with a small file size. Feel free to use this format for a photograph or stock image as long as it contains no text, otherwise the outline of the text will have some pixilation that will give it a faint, fuzzy outline.

PNG and GIF

If you want crisp lines, excellent quality, and/or a transparent background in an image, the PNG or GIF formats are your best choices; just keep in mind that file sizes are going to be large. Either of these formats work wonderfully for graphics that contain any kind of text. Use a PNG format when you have a static image; you can use this format with images that have any number of colors. Logos and vector graphics are also typically saved in PNG. If you want to make an animation and/or have a graphic that is less than 256 colors, use a GIF.

Logo

Description

Your site should always have a logo as it distinguishes your brand and product. In order to upload a logo to your website, we strongly suggest you save it in the PNG format. Your designers may already have prepared a folder for you with your logo saved in various web-enabled formats. If not, then you will need image processing software, such as Photoshop, to save the image in one of the recommended formats.

Placement

Always position your logo at the top of your site, preferably in the header. If you have a side menu, then your logo should be positioned on the top right side of the page.

Proportions

Always use images that are balanced or in proportion. The proportion of an element is well done when there is no unevenness, so the graphic does not look stretched horizontally or vertically. Proportion helps create unity in a design. Elements out of proportion call attention to themselves, sometimes in a way that makes the design look bad. One useful tool that can facilitate image scaling on your page is Bootstrap This JavaScript library scales images responsively, which results in a more mobile-friendly page.
