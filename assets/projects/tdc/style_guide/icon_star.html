Bookmark It

  • About
  • Specs
  • Position
  • HTML

Name

Bookmark It (fa-star)

Description

Icon consists of a solid star. This symbol is associated with functionality that lets users save a component or, more commonly, a link to that component (e.g., the URL of a page) in a directory that they can access at any time. Think of it as a way to store components in the same fashion quick links are saved with the exception that users should be allowed to bookmark multiple components.

Use

Use it when you want to represent:

  • A bookmark
  • Favorite functionality
  • Link to directory with stored components

Example

Sizing

small (16 x 14 pixels)

large (24 x 28 pixels)

Color

AT&T Orange Highlight (#fcb314)


This color indicates a selection has already been made. For example, in a scenario where the icon is being used to bookmark pages, if the user has already clicked on the icon (thus saving the page to his or her bookmarks) it should turn this color.


AT&T Blue (#067ab5)


Use AT&T Blue if you are using the icon as a link that displays all of the previously bookmarked pages. You should use AT&T Blue Highlight when the user hovers over the icon (see Links).


AT&T Gray Highlight (#f2f2f2)


AT&T Gray Highlight should be used if this component is in an "off" or "false" state. For example, if your user is visiting a page that he or she has not bookmarked before, the icon would appear in this color.


General Placement

Position this element in the right side of your header.

Horizontal alignment: center the element horizontally within its container.

Vertical alignment: center the element vertically within the header.
